歸檔頁面

2021年12月3日1 分鐘閱讀 Razon Yang 歸檔 文檔
🎉 Our new modular theme is available now, which ships with much more features and higher flexibility, please consider giving it a try for your new static sites. :)
This theme provides several kinds of layouts, such as posts and docs. Our documentation uses the docs layout. If you're looking for an example that using posts layout, please take a look at the page Markdown Syntax.

目錄

歸檔頁面按照年份對文章進行歸檔。

前置條件

歸檔頁面不會自動創建，因此我們需要先創建一些 Sections：

1$ tree content/archives
2├── 2021
3│   ├── _index.md
4├── 2022
5│   ├── _index.md
6├── _index.md

這些文件只需包含一個 title 屬性即可。

網站參數

名稱類型預設值描述
archiveObject-Archive.
archive.dateFormatStringJan 2Archive date format.
archive.monthFormatStringJanuaryArchive month format, e.g. 1, 01, Jan and January.
archive.basePathString/archives
