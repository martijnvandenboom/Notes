FAQ 页面

2021年12月7日1 分钟阅读 Razon Yang FAQ 文档
🎉 Our new modular theme is available now, which ships with much more features and higher flexibility, please consider giving it a try for your new static sites. :)
This theme provides several kinds of layouts, such as posts and docs. Our documentation uses the docs layout. If you're looking for an example that using posts layout, please take a look at the page Markdown Syntax.

目录

FAQ 页面是一个常见问题的集合。

前置条件

我们需要在 content 目录下创建一个名为 faq/index.md 的页面。

1+++
2title = "Frequently Asked Questions"
3layout = "faq"
4+++

数据

数据存放在 data 目录，其目录结构如下：

1data
2  /en
3    /faq
4      foo.json
5      bar.json
6  /zh-cn
7    /faq
8      foo.json
9      bar.json

如你所看到的，我们按照语言对问题进行归类。每个文件代表一组问题，它们有着以下相同的格式：

 1{
 2  "title": "The group title",
 3  "weight": 1,
 4  "questions": [
 5    {
 6      "question": "The question",
 7      "answer": "The answer"
 8    }
 9  ]
10}
  • weight 参数用于将组按升序排序。
